# Як почати склад з нуля

## 1. Створити адміністратора
```bash
python manage.py createsuperuser
```

## 2. Увійти в систему
Відкрийте сайт, введіть логін і пароль адміністратора.

## 3. Створити ролі
```bash
python manage.py init_roles
```

## 4. Створити користувачів
Створіть користувачів для ролей: адміністратор, комірник, перегляд/аудитор.

## 5. Створити одиниці виміру
Додайте: шт, кг, м, л, упак.

## 6. Створити категорії
Додайте: Електрозапчастини, Механіка, Підшипники, Кабель, Інструмент, Витратні матеріали.

## 7. Створити склади
Додайте: Основний склад, Склад електрика, Склад механіка, Ремонтна зона.

## 8. Створити локації
Додайте: Стелаж 1, Стелаж 2, Комірка A1, Комірка A2, Зона приймання, Зона видачі.

## 9. Створити номенклатуру
Для кожної позиції заповніть назву, внутрішній код, категорію, одиницю виміру та опис.

## 10. Внести початкові залишки
Перерахуйте фактичні залишки і внесіть їх як початковий баланс.

## 11. Далі працювати тільки через операції
Використовуйте: Прихід товару, Видача, Переміщення, Списання, Повернення.

## 12. Не редагувати залишки вручну напряму
Ручна зміна залишків ламає історію. Якщо потрібне виправлення — використовуйте операцію коригування.

## 13. Перевірити залишки
Відкрийте **Залишки** і звірте кількість з фактом.

## 14. Перевірити рухи товарів
Переконайтесь, що всі стартові операції потрапили в журнал рухів.

## 15. Перевірити аналітику
Відкрийте **Аналітика**, задайте період і перевірте прихід, видачу, списання, переміщення та залишки.

## 16. Налаштувати друк етикеток і принтери
Налаштуйте CUPS, додайте принтери, зробіть тестовий друк і перевірте формат етикетки.
