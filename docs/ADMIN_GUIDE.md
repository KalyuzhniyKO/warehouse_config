# Інструкція адміністратора

## 1. Призначення ролі адміністратора
Адміністратор налаштовує користувачів, ролі, довідники, резервне копіювання та контролює якість даних.

## 2. Як створити першого адміністратора

```bash
python manage.py createsuperuser
```

## 3. Як ініціалізувати ролі

```bash
python manage.py init_roles
```

Команда створює групи **Адміністратор складу**, **Комірник**, **Перегляд / аудитор** і призначає базові дозволи.

## 4. Як керувати довідниками
Використовуйте **Керування → Довідники**. Архівуйте непотрібні записи замість фізичного видалення. Архівні записи не пропонуються у формах для нових зв'язків.

## 5. Як знайти дублікати

```bash
python manage.py find_duplicates
```

## 6. Резервне копіювання
Перед масовими змінами або оновленнями створюйте резервну копію бази даних і перевіряйте відновлення на тестовому середовищі.
